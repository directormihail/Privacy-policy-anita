# ANITA — Privacy Policy & Terms of Use

Official legal documentation for **ANITA**, a personal finance application. This repository publishes the **Privacy Policy** and **Terms of Use** that users and app stores can link to for transparency and compliance.

## What it does

These pages explain how ANITA handles personal, financial, and usage data; how banking connections work via Stripe Financial Connections; how analytics (PostHog) is used for product improvement; and the rules governing use of the service. The content is written to align with **GDPR**, **CCPA**, and common app-store requirements.

## Stack

- **HTML5** — semantic structure, English copy  
- **CSS** — embedded styles, responsive layout, accessible typography  
- **No build step** — plain static files you can host on **GitHub Pages**, Netlify, Cloudflare Pages, or any static file server
